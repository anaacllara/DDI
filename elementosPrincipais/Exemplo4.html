Exemplo 4 html - estrututa geral de um documento HTML

Este exemplo mostra a estrutura mínima de um documento html Aqui estão as principais tafs utilizadas:"

  • < !DOCTYPE hmtl >: Define o tipo de documento html, ou seja, a versão do html que será usada na página.
  • < html >: A tag html é o elemento principal do documento html, ela define o container raíz de toda página.
  • < HEAD >: A tag head contém os metadados que serão interpretados pelo navegador. Nela encontramos título da página, links e arquivos para configuração do documento.
  • < BODY >: A tag body é responsável pela exibição do conteúdo visívl do documento, ou seja, tudo o que estiver neste bloco será visível para o usuário.
