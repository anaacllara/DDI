Exemplo 5 HTML - tags de seção e títulos

Este exemplo mostra como usar as tags de título <h1> à <h6> para organizar o conteúdo do documento em seções hierárquicas.

Descrição

As tags de título são usadas para definir cabeçalhos e subtítulos. Elas variam de <h1>(mais imoportante) à <h6> (menos importante).
