Exemplo 6 html - Uso das tags <p>, <hr>, <br>, <strong>, <i>, <em>

Este exemplo mostra o uso das tags de formatação de texto bem como a sua organizações visual.

Descrição das tags abordadas

  • <p> Define um parágrafo
  • <hr> Cria uma linha horizontal para separar conteúdos
  • <br> Insere uma quebra de linha no documento html
  • <strong> Define o texto em negrito (com importância semântica)
  • <i> Define o texto em itálico (com importância semântica).
  • <em> Define texto enfatizado e o leitor de tela pode ler com ênfase verbal.
