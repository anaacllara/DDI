Exemplo 10 HTML - Tabelas

As tabelas em HTML são representadas pela tag <table> que são úteis apra exibir dados tabulares, tais como informações em linhas e colunas.

Descrição dos principais atributos para criação de tabelas em HTML

  • <table>: Esta é a tag principal que envolve toda a tabela html
  • <tr>: Esta tag representa uma linha da tabela. E nela termos as tags <th> e <td>
  • <th>: Esta tag representa o cabeçalho da tabela, <th> é o acrônimo de table header.
  • <td>:A tag <td> (table data) representa a célula da tabela.
  • <theader>: Tag semântica que representa o cabeçalho. Ela é utilizada juntamente com a tag <th>
  • <tbody> Tag semâmtica que representa o corpo da tabela. Nela, geralmente, usamos a tag <td>
  • <tfoot>: Tag semântica que representa o rodapé da tabela.
