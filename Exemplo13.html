Exemplo 13 HTML - Formulários

Os formulários HMTL sõa uma das principais formas de se interagir com aplicaões web. Esses formulários utilizam diversos componentes que melhorarão a experiência do usuário.

Para trabalhar com formulários devemos usar a tag <form>em conjunto com outras tags que vão representar os componentes de fómulários, tais como: labels(rótulos), inputs(caixa de texto), botões, elementos de checagem, de seleção e entre outros componentes. Neste exemplo, demonstraremos o uso dos métodos Get e Post

Métodos do Protocolo HTTP

Get

Quando trabalhamos com a criação de formulário HTML, ,nós deveremos, na maior parte das vezes, utilizar o protocolo de rede HTPP, esse protocolo possui vários métodos que são utilizados para facilitar o processo de comunicação entre o cliente e o sevidor.

O método GET é aquele que anexa os dados do fomrulário a uma URL, esses dados dos fomrulário são visíveis na barra de endereço do navegador.

É importante saber que este método não é recomendado para operações de inclusão, pois os dados vão aparecer para qualquer pessoa, o que pode ser inseguro.

Post

Já o método POST é aquele que envia os dados para um servidor no corpo da página. Essas informações não são visíveis para o cliente, sendo um pouco mais seguro que quando utilizamos o método GET.
